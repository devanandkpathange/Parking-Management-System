# Parking Management System

A full-stack parking management system developed to streamline vehicle parking during large-scale campus events. The system improves parking allocation, reduces congestion, and supports valet-assisted parking for efficient vehicle management.

---

## Problem Statement

Managing vehicle parking during college events can become difficult due to high traffic and limited parking availability. This project provides an organized solution by allocating parking slots, blocking occupied spaces, and assisting users through a valet workflow.

---

## Features

- Parking Slot Allocation
- Real-Time Slot Availability
- Parking Slot Blocking
- Valet Management Workflow
- User-Friendly Interface
- Event-Based Parking Management
